隐私政策

更新日期：2026年5月4日
感谢你访问高新炀的小站。本隐私政策将说明本站如何收集、使用及保护你的信息。

本站不会出售、转让或恶意泄露你的个人信息。所有数据收集仅用于改善体验、了解访问趋势以及回复你的留言。

一、我们收集的信息

在你访问本网站的过程中，可能会产生以下几类信息：

  • 自动收集的访问数据（匿名统计）： 通过 不蒜子（busuanzi） 和 Umami Analytics (云服务) 收集匿名的访问量、页面浏览量、访客数量、来源地区、设备类型等聚合数据。这些数据无法识别你的个人身份，仅用于了解网站受欢迎程度及内容优化。
  • 评论互动数据： 当你使用 Twikoo 评论系统 发表留言时，会按照系统要求提供昵称、邮箱（可选）、网站（可选）以及评论内容。这些数据将展示在对应评论区，以便站长回复及社区交流。Twikoo 可能会记录你的 IP 地址用于反垃圾，但站长不会主动公开你的 IP。
  • 本地存储与浏览器缓存： 在获得你的同意后（通过 Cookie 同意横幅），本站会将以下信息存储在浏览器的 localStorage 中：
    • 主题偏好（亮色/暗色模式）；
    • 文章/作品列表缓存（提升加载速度）；
    • 阅读滚动位置、搜索筛选条件、友链缓存等体验优化数据；
    • “欢迎回来”模态框的访问记录、网站版本记录。
  • 必要的会话记录： 当你使用外链跳转确认、页面导航等交互时，不会主动记录个人敏感信息。

重要说明： 本站 从不收集 你的密码、真实身份证号、银行账户等敏感信息。所有本地存储数据均保留在你的设备中，你可以随时通过浏览器设置清除。

二、Cookie 与本地存储的使用

本站严格遵循 “透明且可拒绝” 的原则。当你首次访问本站时，页面底部会弹出 Cookie 同意横幅：

  • 同意： 表示你允许网站使用 localStorage 存储主题偏好、缓存数据、阅读位置等功能，以换取更流畅的个性化体验。同时，非必要的统计脚本（如 Umami）会正常运行。
  • 拒绝： 本站将禁用几乎所有的本地存储功能（StorageController 进入禁用模式），并且已经存储的相关数据会被清除。你仍然可以正常阅读文章、浏览作品，但无法保存主题/搜索条件等个性化设置，匿名统计依然基于基础访问量计数（不蒜子遵循极简原则）。

注意：“不蒜子”计数仅使用 JavaScript 单次请求，不依赖 Cookie 标识个人；Twikoo 评论系统依赖浏览器本地存储保存用户信息（如昵称、邮箱），但仅在提交评论时激活，并遵循 Twikoo 自身隐私逻辑。你可以在任何时候清除浏览器的 localStorage 和 Cookie 来撤回授权。

三、第三方服务及其隐私说明

本网站集成了以下第三方服务，它们分别遵循各自隐私政策：

  • 不蒜子 (busuanzi.ibruce.info) —— 极简访客计数器，不设置 Cookie，仅通过向 https://cdn.busuanzi.cc/ 发送匿名请求统计 PV/UV。数据仅用于展示全站访问量，无用户画像。
  • Twikoo 评论系统 —— 开源评论后端 (部署于 Netlify Functions)。你提交的昵称、邮箱、网址及评论内容将被存储于 Twikoo 数据库中。站长可管理评论（删除/审核）。Twikoo 不会将数据用于营销。更多信息可参考 Twikoo 官方说明 。
  • Umami Analytics —— 轻量级匿名分析工具，不采集个人身份信息，仅记录页面浏览行为（如访问路径、浏览器版本）。Umami 遵守 GDPR / ePrivacy 规范，数据所有者为网站站长。
  • Font Awesome / 外部字体图标 —— 通过 CDN 加载，可能记录你的 IP 地址用于 CDN 分发，但本网站无法控制此类基础网络日志。

此外，文章中的外部链接（例如 GitHub、Bilibili 等）会通过 外链跳转确认模态框 提醒你即将离开本站，并展示目标域名。该功能不记录你的跳转行为。

四、数据安全与保留期限

  • 本地存储数据： 完全保存在你的浏览器中，你可随时通过开发者工具或浏览器设置手动清除。
  • 评论数据： 除非你主动联系站长请求删除，否则你的评论内容将永久保留在该篇文章下方（便于社区交流）。但你可通过邮箱联系 gao_xinyang@foxmail.com 或通过留言板提出删除申请，站长会在合理时间内处理。
  • 统计数据： 不蒜子和 Umami 的数据聚合周期均为长期保留，但无法追溯到具体个人。

本站通过 HTTPS 传输数据（若服务器支持），尽可能保障通信安全。但由于互联网并非绝对安全，请谨慎不要提交私密信息到公开评论区。

五、您的权利

基于数据保护普遍原则，你在使用本网站期间享有以下权利：

  • 知情权： 本隐私政策已清晰告知数据处理范围。
  • 撤回同意： 你可以随时通过浏览器清除 localStorage 和 Cookie（尤其是 cookieConsentAccepted 等键值），刷新页面后 Cookie 同意横幅会重新出现，届时可选择拒绝。
  • 拒绝跟踪： 虽然本网站没有植入广告追踪，但你可以在浏览器启用“请勿跟踪”(DNT) 选项，Umami 会尊重该偏好。
  • 删除评论： 若你希望删除自己曾发布的评论，请联系站长并附上评论所在页面及昵称（或邮箱）。
如需行使任何权利或有隐私疑虑，欢迎发送邮件至 gao_xinyang@foxmail.com。

六、未成年人保护

本网站内容偏向技术分享、个人随笔及作品展示。若你未满 14 周岁，请在父母或监护人指导下使用评论功能，切勿在评论区透露隐私信息。我们不会刻意收集未成年人信息，一旦发现监护人可联系我删除相关评论。

七、隐私政策的更新

本页面会随网站功能或法律法规的变动进行修订。重大变更（如新增数据收集类型）将通过网站首页公告或 Cookie 横幅提示。修订后的政策自发布之日起生效。建议你定期查阅本页面以确保了解最新内容。
